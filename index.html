A11yWrapper Library

This slider is purely built on JavaScript and fully customizable. Developer needs to call the wrapper class and show container where slider will be created on the fly.

Create fast, accessible components with less code

This is a purely JavaScript based library with no dependency to third party libraries. This will act as "Plug n' Play" on your project. Leverage the components from this library to create complex HTML elements and forget about making it accessible. Library will take care of it.

Installation

At first download A11yWrapper.CSS and A11yWrapper.JS

Include following files to the webpage and you're good to go.

<link rel="stylesheet" href="{location of css file}/A11yWrapper.css" />

A11yWrapper.JS

<script src="{location of JS file}/A11yWrapper.min.js" type="text/javascript"></script>
